## Troubleshooting

**Issue:** "Can't verify my bank account"
- **Solution:** Check with your bank, sometimes takes 2-3 business days

**Issue:** "Payment declined when testing"
- **Solution:** Use test mode first, or use test credit card: 4242 4242 4242 4242

**Issue:** "Customers can't access content"
- **Solution:** Make sure Google Drive/Doc links are set to "Anyone with link can view"

**Issue:** "Not getting paid out"
- **Solution:** Check payout threshold ($10 minimum), verify payment method

**Issue:** "High churn rate (cancellations)"
- **Solution:** 
  - Add more value (more wallpapers, more affirmations)
  - Engage in Discord community
  - Send exclusive premium-only emails
  - Ask for feedback
