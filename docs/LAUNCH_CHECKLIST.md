## Common Issues

**Site not updating after push:** Wait 2–3 minutes. Check GitHub Settings → Pages to confirm deployment.

**Card counter stuck at 0:** It fires once per card created in `send-card.html`. Check browser console for fetch errors.

**MailerLite not capturing emails:** Verify the API key in `send-card.html` is current. Rotate at MailerLite → Integrations → API if needed.
