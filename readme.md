## 🚀 Quick Start Guide

1. **Download** and extract all template files
2. **Open** `index.html` in your browser to see the template
3. **Edit** HTML files with your brand content and company information
4. **Customize** colors through CSS variables in `styles.css`
5. **Replace** placeholder images with your own assets
6. **Upload** to your hosting provider

## 🎨 Customization

### Colors
Edit CSS variables in `styles.css` (around line 12):
```css
:root {
    --color-primary: #6366f1;     /* Main brand color */
    --color-secondary: #14b8a6;   /* Accent color */
    --color-bg-dark: #0f172a;     /* Background color */
    --color-text: #e2e8f0;        /* Text color */
    /* Many more variables available */
}
```

### Content Updates
- Replace "Your Company" with your business name throughout all HTML files
- Update contact information in contact.html and footer sections
- Customize hero title, subtitle, and feature descriptions
- Modify testimonials and statistics to match your business

### Forms
- Integrates with Tally.so forms out of the box
- Update the `formId` in the JavaScript section of index.html
- Can be easily modified for other form services (ContactForm7, etc.)
